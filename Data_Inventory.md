## R Package Dependencies

See `Workshop_Setup_Guide.md` for the complete package list with versions and
a full installation script (`install_packages.R`). Quick summary:

### Bioconductor packages (tutorial core)
- `dada2` (DADA2 pipeline)
- `phyloseq` (microbiome data structure and analysis)
- `decontam` (contaminant identification)

### Bioconductor packages (extended pipeline)
- `DESeq2`, `ShortRead`, `Biostrings`, `DECIPHER`, `genefilter`

### R-universe packages
- `microViz` (extended phyloseq utilities)

### CRAN packages (tutorial core)
- `tidyverse` (includes ggplot2, dplyr, tidyr, stringr, purrr, readr, forcats, tibble)
- `vegan` (community ecology: distances, PERMANOVA)
- `data.table` (efficient data manipulation for DA section)

### CRAN packages (extended pipeline)
- `ade4`, `phangorn`, `lmerTest`, `car`
- `RColorBrewer`, `cowplot`, `ggpubr`, `ggsignif`, `patchwork`, `scales`, `gt`
- `furrr`, `future`, `jsonlite`

### Quick install
```r
if (!requireNamespace("BiocManager", quietly = TRUE))
    install.packages("BiocManager")
BiocManager::install(c("dada2", "phyloseq", "decontam",
                        "DESeq2", "ShortRead", "Biostrings",
                        "DECIPHER", "genefilter"))
install.packages("microViz",
  repos = c(davidbarnett = "https://david-barnett.r-universe.dev",
            CRAN = "https://cloud.r-project.org"))
install.packages(c("tidyverse", "vegan", "data.table", "knitr", "rmarkdown",
                    "ade4", "phangorn", "lmerTest", "car",
                    "RColorBrewer", "cowplot", "ggpubr", "ggsignif",
                    "patchwork", "scales", "gt",
                    "furrr", "future", "jsonlite"))
```

## Distribution Notes

For distributing to workshop participants:

1. **Minimum distribution** (~200 MB): Raw FastQ files + `rds/` + Part 2 phyloseq +
   metadata file + the Rmd. Exclude SILVA ref databases and have participants
   use pre-computed taxonomy.

2. **Full distribution** (~320 MB): Include SILVA ref databases so participants
   can optionally run `assignTaxonomy()` on their own.

3. **Compressed archive**: The entire tutorial directory compresses to ~150 MB
   as a zip file.
