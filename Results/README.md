# Results Directory

Created by: Michael Sieler  
Last updated: 2026-03-06

This directory stores analysis outputs generated by the workshop tutorial.
It is organized by workflow stage so files are easy to find and regenerate.

## Structure

- `DADA2/`
  - `Figures/` - Part 1 figures (quality plots, filtering diagnostics, etc.)
  - `Tables/` - Part 1 exported tables (filter summary, read tracking, metadata after filtering)
- `MicrobiomeAnalysis/`
  - `Figures/` - Part 2 figures (composition, alpha/beta diversity, differential abundance)
  - `Tables/` - Part 2 exported tables (alpha diversity, PERMANOVA, DA result tables)

## Notes

- If this directory appears empty, the tutorial has not been run yet in this environment.
- Outputs are produced by `Code/Microbiome_Workshop_Tutorial.Rmd`.
- Intermediate checkpoint `.rds` files are stored in `Data/DADA2` and
  `Data/MicrobiomeAnalysis`, not in `Results`.
